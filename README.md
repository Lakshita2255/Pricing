# Credex AI Spend Audit — 2-Day Starter

This repository contains the first 2 days of a Credex-style AI spend audit app. It is intentionally a partial build: the focus is on the early product scaffolding, the spend input form, state persistence, and a basic reasoning stub for audit recommendations.

## What is included

- React + Vite + TypeScript starter app
- Spend input form for AI tools
- Local browser persistence for form state
- Basic audit result summary and per-tool recommendation logic
- Project documentation for the first 2 days of work

## What is intentionally left for later

- Full audit engine covering all required vendors
- Shareable public URLs and Open Graph metadata support
- Lead capture and backend storage
- Transactional email confirmation
- Complete pricing data references for every tool
- End-to-end automated tests

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the app in your browser at the URL shown by Vite.

## Why this scope

This is the work that fits into the first 2 days of a 7-day assignment. The app is structured so the remaining 5 days can focus on completing the audit engine, shareable reporting, lead capture, production deployment, and documentation for interviews and GTM.
